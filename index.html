Expense Tracker

A CLI-based expense tracker that logs daily or monthly spending with categories like food, rent, books, and entertainment.

How It Works

                
    
                    
  •  Initialize the Tracker: Create a new instance of the `ExpenseTracker` using `ExpenseTracker::new()`.
    • 
                    
  •  Add Expenses: Use the `add_expense` method to add a new expense. Provide a category (e.g., "food") and an amount.
    • 
                    
  •  View Report: Call the `view_report` method to see a summary of all expenses categorized.
    • 
                    
  •  Export to CSV: Use the `export_to_csv` method to export the expenses to a CSV file for easy sharing and record-keeping.
    • 
                    
  •  Interactive CLI: The main function runs a loop allowing users to choose from options to add an expense, view a report, export to CSV, or exit the program.
